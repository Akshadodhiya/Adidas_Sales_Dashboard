# Adidas Data Dashboard (Python + Streamlit)

## 📊 Project Overview
This project is an interactive data dashboard created using Python and Streamlit.  
The dashboard helps in understanding data through charts and tables.  
It allows users to view insights and analyze performance using visualizations.

## 🚀 Features
- Interactive charts and graphs
- Easy to understand dashboard layout
- Data visualization for better decision making
- User friendly interface

## 🛠️ Tools & Technologies Used
- Python
- Streamlit
- Pandas
- Matplotlib / Plotly

## ▶️ How to Run the Project

1. Download or clone this repository
2. Install required libraries
3. Run the project
4. Open in browser

## 📸 Output
Dashboard displays charts and data insights.

## 📌 Purpose
This project demonstrates basic data analysis and dashboard creation skills for Business Analyst role.
